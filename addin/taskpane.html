A fully agentic AI co-pilot that reads your workbook, understands your data, and autonomously takes action — all from a single chat command.
Capabilities
📖Reads entire workbook — all sheets, values, formulas, structure
✍️Writes values, sets Excel formulas, pastes data from files
🎨Formats cells — bold, colors, borders, number formats
📊Creates charts (bar, line, column, pie) from data ranges
📋Adds, renames, and manages worksheets
🔃Sorts and filters data ranges
📎Reads attached PDF, CSV, XLSX, TXT, JSON files
🤖Multi-step agent loop — plans, executes, verifies, reports
🔌OpenRouter · Gemini · Groq · Claude · Qwen/Alibaba · Ollama (local)
Quick Start
1. Go to Settings → pick your AI provider → enter API key → Save
2. Click "Test Connection" to verify
3. Come back to Chat → type your command
4. Watch the agent work step by step

Example commands:
• "Summarize the data in Sheet1 and add a total row"
• "Create a bar chart of column A vs column B"
• "Format all cells in row 1 as bold with green background"
• "Read the attached CSV and paste it into Sheet2 starting at A1"
• "Add a new sheet called Summary and write key metrics there"
